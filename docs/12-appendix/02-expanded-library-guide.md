# 扩展资料库说明

如果你希望把这套教程继续扩成更大体量，我已经预留了自动生成的内容库入口。

## 生成命令

```bash
node scripts/generate-expanded-docs.js
```

## 生成位置

会输出到：

- [generated/expanded-docs](generated/expanded-docs)

## 目前会生成什么

- 100 条术语讲解
- 100 条新手问答
- 100 条排错案例
- 100 条实操任务
- 100 条部署变体

## 为什么这样设计

因为真正“超大体量教程”不能只靠一篇超长 Markdown。更好的做法是：

1. 用人工主教程保证质量
2. 用扩展卷保证覆盖面
3. 后续继续按主题叠加

## 你怎么看待“百万字”

这里我给的是“可扩展到大体量”的仓库结构，而不是盲目塞一篇没人会读完的超长单文档。这样更适合：

- 真正学习
- 真正维护
- 真正后续继续扩写
